# WhoIsIn — Сценарий видео-демо (2 мин)

> Открой http://185.46.11.151 в браузере перед записью.
> Рекомендуемый инструмент: OBS / QuickTime Screen Recording.
> Разрешение: 1920x1080, браузер на весь экран.

---

## Сцена 1: Главная страница (0:00–0:10)

**Что на экране:** Список мероприятий (Events)

**Действия:**
1. Открыть http://185.46.11.151
2. Пауза 2 сек — показать UI (дизайн-система, Swiss Minimalism)
3. Нажать **"Новое мероприятие"** (кнопка "+" или "Create Event")

---

## Сцена 2: Создание мероприятия (0:10–0:25)

**Что на экране:** Модалка / форма создания

**Действия:**
1. Ввести название: `Demo Video`
2. Опционально: ввести описание
3. Нажать **"Создать"**
4. Автоматически откроется страница мероприятия (пустая)

---

## Сцена 3: Загрузка фото (0:25–0:45)

**Что на экране:** Страница мероприятия с UploadZone

**Действия:**
1. Подготовить заранее 10–15 фото с лицами (можно из TCF)
2. Перетащить (drag & drop) фото в зону загрузки
3. Показать как фото появляются в списке
4. Пауза — показать что все загружены

---

## Сцена 4: Запуск pipeline (0:45–1:15)

**Что на экране:** Кнопка "Run Pipeline" → real-time прогресс

**Действия:**
1. Нажать **"Run Pipeline"** (или "Запустить обработку")
2. **КЛЮЧЕВОЙ МОМЕНТ** — показать real-time прогресс через WebSocket:
   - Фаза 1: Detecting faces...
   - Фаза 2: Generating embeddings...
   - Фаза 3: Clustering faces...
   - Фаза 4: Generating descriptions...
   - Фаза 5: Done!
3. Дождаться завершения (зелёная галочка)

---

## Сцена 5: Галерея персон (1:15–1:30)

**Что на экране:** Persons grid — карточки с лицами

**Действия:**
1. Прокрутить вниз к секции персон
2. Показать карточки: превью лица, кол-во фото, имя
3. Кликнуть на одну персону — откроется PersonPage
4. Показать все фото этого человека
5. Показать кнопку **ZIP download** (скачать все фото)

---

## Сцена 6: Поиск по селфи (1:30–1:50)

**Что на экране:** SelfieSearch компонент

**Действия:**
1. Вернуться на страницу мероприятия
2. Найти секцию **"Найти себя"** (Selfie Search)
3. Загрузить селфи (фото лица одного из людей на загруженных фото)
4. Показать результат: **top-3 совпадения** мгновенно
5. Кликнуть на совпадение — перейти к фото

---

## Сцена 7: Финал (1:50–2:00)

**Что на экране:** Обзор результата

**Действия:**
1. Вернуться на главную — показать мероприятие со статистикой
2. Пауза на 3 сек

---

## Чек-лист перед записью

- [ ] http://185.46.11.151 открывается
- [ ] Подготовить 10–15 фото с лицами (разные люди, несколько фото каждого)
- [ ] Подготовить 1 селфи для поиска (должно быть лицо человека из набора)
- [ ] Закрыть все вкладки, уведомления, панель закладок
- [ ] Разрешение экрана 1920x1080
- [ ] Браузер в полноэкранном режиме (Cmd+Shift+F в Chrome)
- [ ] Если TCF мероприятие уже есть — можно использовать его вместо нового
