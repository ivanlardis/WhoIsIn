# UI/UX Design — WhoIsIn

## Design System

### Стиль
Modern minimal с акцентом на фотоконтент. Чистый, воздушный интерфейс — фото должны быть в центре внимания, UI не должен отвлекать.

### Цветовая палитра
- **Primary:** Indigo-600 (`#4F46E5`) — кнопки, акценты, прогресс
- **Background:** Slate-50 (`#F8FAFC`) — основной фон
- **Surface:** White (`#FFFFFF`) — карточки, панели
- **Text:** Slate-900 (`#0F172A`) — основной текст
- **Muted:** Slate-500 (`#64748B`) — вторичный текст
- **Success:** Emerald-500 (`#10B981`) — завершение, совпадения
- **Destructive:** Red-500 (`#EF4444`) — удаление, ошибки

### Типографика
- **Заголовки:** Inter (semibold)
- **Текст:** Inter (regular)
- **Моноширинный:** JetBrains Mono (метрики, confidence)

### Компоненты (shadcn/ui)
Card, Button, Dialog, Progress, Badge, Avatar, Tabs, DropZone, Skeleton

## Responsive

- **Desktop (>1024px):** 4-5 колонок карточек персон, боковая панель на странице мероприятия
- **Tablet (768-1024px):** 3 колонки, collapsed sidebar
- **Mobile (<768px):** 2 колонки, bottom navigation, selfie search — полноэкранный
